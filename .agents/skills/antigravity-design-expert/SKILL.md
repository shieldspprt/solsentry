# Antigravity Design Expert Standards

A comprehensive design system skill for crafting high-tier, legible, and visually stunning web interfaces.

## 1. Visual Hierarchy & Typography
- Enforce explicit typography scales (minimum 14px body, 16px section headers, 24px+ hero metric titles).
- Prioritize high text contrast against dark backgrounds using slate-100 for primary headings, slate-300 for labels, cyan-300 for active states, and emerald-300 for status badges.
- Avoid text truncation or text wrapping in tables by keeping labels concise and self-descriptive.

## 2. Spacing & Layout Architecture
- Maintain generous breathing room between layout containers using grid gap-6 to gap-8 and vertical space-y-8.
- Apply spacious padding on card containers (minimum p-6 / p-7) and table cells (minimum px-6 py-5).
- Ensure navigation touch targets exceed 44px minimum height.

## 3. Micro-Animations & Interaction
- Add smooth color transition-all classes with duration-200 on all interactive elements.
- Implement subtle glassmorphism backdrops (`backdrop-blur-md`) for sticky header bars and navigation overlays.
