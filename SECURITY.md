# Security Policy

SolSentry is a security-first risk engine and guardrail middleware for Solana AI agents. We take security vulnerabilities seriously.

## Reporting a Vulnerability

If you discover a security vulnerability in SolSentry, please do **NOT** open a public issue on GitHub.

Instead, please report the vulnerability confidentially:

- **Email**: `security@solsentry.io`

Please include:
- A description of the issue and potential impact
- Step-by-step instructions or proof-of-concept code to reproduce
- Affected components or versions

## Response Timeline

- **Acknowledgement**: Within 24 hours
- **Assessment & Triage**: Within 48 hours
- **Patch & Release**: Priority fixes deployed within 7 business days

Thank you for helping keep the Solana AI agent ecosystem secure!
